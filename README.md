# simple-crud-api

```
# Routes
GET      /person
GET      /person/:id
POST     /person
PUT      /person/:id
DELETE   /person/:id
```

## Usage

```
# Install dependencies
npm i

# Run in develpment
npm run start:dev

# Run in production
npm run start:prod
```

## Tests

***Для успешного завершения тестов необходим работающий экземпляр API на localhost (запустить API в другом терминале)***

### Terminal 1
```
# Run api
npm run start:dev || npm run start:prod
```

### Terminal 2
```
# Run api
npm run test
```
